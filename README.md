# SurfGuard

SurfGuard is a gentle Chrome extension that helps you remember why you opened a site — without blocking it.

SurfGuardは、サイトをブロックせずに「何をしに来たか」と「どれくらい滞在しているか」をそっと思い出させるブラウザ見守りツールです。

## Status

SurfGuard is currently in beta.

This version focuses on the core browsing companion features:

- Purpose selection
- Gentle reminders
- Breaks
- Local-only browsing awareness

SurfGuardは現在ベータ版です。  
このバージョンでは、目的選択、やさしいリマインダー、休憩、ブラウザ内だけで完結する見守り機能を中心に提供しています。

## Privacy

SurfGuard does not read page content, posts, video titles, form inputs, private messages, or passwords.

Data is stored locally in the Chrome browser and is not transmitted to external servers.

SurfGuardは、ページ本文、投稿内容、動画タイトル、入力フォーム、個人メッセージ、パスワードを読み取りません。  
データはChromeブラウザ内に保存され、外部サーバーへ送信されません。

See: [Privacy Policy](./privacy.html)
